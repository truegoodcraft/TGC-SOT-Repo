## About This Approach

These documents use **Source of Truth (SoT) discipline**:

✅ **Versioned** - Every change gets a version bump + changelog entry
✅ **Authoritative** - Where code and docs disagree, docs must be updated
✅ **Explicit** - Anything not stated is "not specified"
✅ **Durable** - Designed to outlive any single tool or platform

This is the same system used to build BUS Core itself.
